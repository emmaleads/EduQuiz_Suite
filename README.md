# EduQuiz Assessment Suite

A lightweight, multi-department tool designed for faculty members to parse exam questions from Word documents into structured Excel files and simulate interactive test environments.

## 🚀 How to Run the App (No Python Setup Required)

1. Click the green **Code** button at the top right of this page and select **Download ZIP**.
2. Extract the downloaded ZIP file completely to a folder on your computer.
3. Open the extracted folder and double-click **`Launch_EduQuiz.bat`**.

*Note: On the first launch, the script will automatically configure Python and the necessary libraries. This may take 1–2 minutes. Subsequent launches will open instantly.*

## 🛠️ Requirements
* Windows 10/11
* Active internet connection (on first launch only)
